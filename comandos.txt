-> Configura o pc com nome e e-mail do responsável pelo código
git config --global user.name "Euller Lourenço"
git config --global user.email "meuemail@gmail.com"

-> Inicializa um repositório git
git init 

-> Verificar o status do repositório
git status

-> Adicionar os novos arquivos no git para versionar
git add . 

-> Salva uma versão dos arquivos
git commit -m "Meu primeiro commit"

-> Enviar arquivos locais para nuvem
git push

-> Clonar o repositório
git clone (endereço do repositório git)

-> Baixar atualização da nuvem do github para uma pasta já existente
git pull
